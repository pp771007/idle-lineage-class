# 放置天堂(idle-lineage-class）

天堂（Lineage）主題的網頁放置遊戲。打怪、撿裝、強化、攻城，掛著就會自己變強。

## ▶ 立即遊玩

### 👉 https://pp771007.github.io/idle-lineage-class/

用電腦或手機的瀏覽器點開網址就能玩，不用安裝、不用註冊。

## 加了哪些東西

這個版本在原作之上多做了幾個方便的功能：

- **🌙 離線掛機** — 把瀏覽器關掉、人離開，回來時也會自動結算這段時間掛機賺到的金幣、經驗和掉落（最多算 24 小時）。
  > 想離線多賺，記得先把角色裝備和自動補血/補藥設定好，能在那張地圖撐得住才行；撐不住就會在掛機中陣亡、收益只算到陣亡前。

- **📱 手機也能玩** — 用手機開網址就是手機專屬版面：下方有「戰鬥／設定／背包／日誌」切換，日誌做成可開關的浮動視窗，物品視窗也不會再爆出畫面。點按鈕（喝水、撿到東西等）時，畫面上方還會浮一則提示，讓你知道剛剛發生了什麼。

- **🔍 怪物 / 掉落查詢** — 首頁可以搜尋怪物、地圖、掉落物，查牠的數值、會出現在哪些地圖、各物品的掉落機率。也能用 ↗ 開成獨立分頁，邊玩邊查。

- **📚 小百科** — 首頁入口，把職業專精、武器特性、職業魔法、任務、套裝、強化、負重、席琳的世界、血盟、傲慢之塔共十個主題整理成查詢頁，還能用關鍵字一次搜尋（例：出血、套裝、屠龍劍），不用自己土法煉鋼。同樣可以用 ↗ 開成獨立分頁。

## 關於存檔

進度是存在「你目前用的這個瀏覽器」裡的。換瀏覽器、換裝置，或本機開的檔案版與網路版，都是**各自獨立的存檔**。要把進度帶過去，請用遊戲內建的「存檔匯出 / 匯入」功能（主選單與「設定」頁都有入口）。
