# idle-lineage-class

天堂(Lineage)主題的網頁放置遊戲,加上自製外掛功能。

## 原作者 / 出處

遊戲本體(`index.html` 與 `assets/`)由 **巴哈姆特 秋玥** 製作:

> https://forum.gamer.com.tw/C.php?page=1&bsn=84452&snA=8362&tnum=273

本專案在原作之上,以「**外掛 JS**」的方式加入額外功能,**不修改原作者的遊戲程式碼**(只在 `index.html` 的 `</body>` 前加上引用外掛的 `<script>`)。原作者更新版本時,把那幾行 `<script>` 重新貼回去即可接上。

## 外掛功能

| 檔案 | 功能 |
|---|---|
| `afk-offline.js` | **離線掛機**:關掉瀏覽器也會結算掛機收益(上限 24 小時、撞死即停、依存檔位 1/2/3 分開、多分頁安全) |
| `afk-mobile.js` | **手機版面**:底部導覽列(戰鬥/設定/背包/日誌)、一行式狀態列、可開關的浮動日誌面板、修正物品彈窗在手機上溢出 |
| `afk-savedata.js` | **存檔匯入/匯出**:整包 localStorage 備份/搬家,可存成檔案或複製文字 |

## 玩法

用瀏覽器開啟 `index.html` 即可遊玩(本機 `file://` 也可)。

## 開發規則

所有自訂功能一律以獨立外掛 JS 實作,不動原作者程式碼;詳見 [`CLAUDE.md`](CLAUDE.md)。
新增外掛時務必同時在 `index.html` 補上對應的 `<script>` 引用。
